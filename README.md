## Getting Started

You need to have installed NodeJs 



### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/matheuslima99/devLinks
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. To run on your device

   ```sh
   npm start 
   ```
